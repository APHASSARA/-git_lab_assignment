# Git Lab Assignment

## What I Learned
In this assignment, I learned how to use Git and GitHub to manage a project. I learned how to initialize a repository, track changes, create branches, merge them, and use pull requests. This helped me understand how version control works in real-world development.

## Challenges Faced
One of the challenges I faced was resolving merge conflicts. It was confusing at first because of the conflict markers, but I learned how to fix them by editing the file manually. Another challenge was understanding the correct sequence of Git commands.

## How Git Helps in Real Projects
Git helps developers manage their code efficiently by tracking changes and allowing multiple people to work on the same project. It also helps prevent data loss and makes collaboration easier through branching and pull requests.
